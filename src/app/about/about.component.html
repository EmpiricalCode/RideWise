Explore a Whole New World of Cycling With RideWise.

by Daniel Long

Simply set a starting point, ending point, and hit the go button. RideWise will display a biking route, as well as provide data on weather, how long the ride will take, and elevation.

RideWise is a web application written in AngularJS with the goal of making bike rides as easy as possible to plan. It uses WeatherAPI for any weather related information, open-elevation for altitude information, and the TomTom API for the map and routing calculation. Make sure to check out the Github repository!
